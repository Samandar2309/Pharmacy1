# Dorixona Frontend

Modern onlayn dorixona frontend (React + Vite + TailwindCSS)

## Ishga tushirish

```bash
npm install
npm run dev
```

Backend: http://127.0.0.1:8000
Frontend: http://localhost:3000

## Features

- ✅ Authentication (Login/Register)
- ✅ Product listing with search & filter
- ✅ Shopping cart
- ✅ Responsive design
- ⚠️ Checkout, Orders, Profile (coming soon)

## Tech Stack

- React 18
- Vite
- TailwindCSS
- Zustand
- Axios
- React Router DOM
